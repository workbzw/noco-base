# 数据表

开发一个系统之前，我们通常要对业务进行抽象，建立数据模型。在 NocoBase 里，数据表的概念与关系型数据库的数据表概念相近。

点击“数据表配置”按钮，进入数据表配置界面。在这里，可以对数据表进行新增、编辑、删除等操作。

## 字段类型

- 基本类型
    - 单行文本
    - 多行文本
    - 手机号码
    - 电子邮箱
    - 数字
    - 百分比
    - 密码
    - 图标
- 选择类型
    - 勾选
    - 下拉菜单（单选）
    - 下拉菜单（多选）
    - 单选框
    - 复选框
    - 中国行政区
- 多媒体
    - 附件
    - Markdown
    - 富文本
- 日期 & 时间
    - 日期
    - 时间
- 关系类型
    - 关联
    - 子表格
- 系统信息
    - ID
    - 创建日期
    - 最后修改日期
    - 创建人
    - 最后修改人
